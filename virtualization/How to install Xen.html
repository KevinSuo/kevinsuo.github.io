How to install Xen

 

Environment:

Linux environment: CentOS 6.5(Linux kernel 3.14)

Xen source code: Xen 4.5.0

 

 

Step:

 

1, if your Linux kernel is lower than 3.0, change your kernel to 3.xx or higher version

 

install the new Linux kernel:

# wget https://www.kernel.org/pub/linux/kernel/v3.x/linux-3.14.48.tar.xz

# xz -d linux-3.14.48.tar.xz

# tar xvf linux-3.14.48.tar

# cd linux-3.14.48

# make menuconfig

 

open the following options:

Processor type and features  --->

Linux guest support  --->

Open All options

 

 

 

 

 

Device Drivers  --->

Block devices  --->

Open the following two options:

Xen virtual block device support

Xen block-device backend driver

 

 

 

 

 

Device Drivers  --->

Xen driver support  --->

Open all options

 

 

 

Device Drivers  --->

Network device support  --->

Open the following two options:

Xen network device frontend driver

Xen backend network device

 

 

 

Save and exit, you will find a new .config file in the folder:

 

 

compile and install the new linux kernel:

# make

# make modules

# make modules_install

# make install

 

modify boot files to run with new kernels:

# vim /boot/grub/grub.conf

 

change the default boot option as 0

 

 

reboot the machine

# reboot

 

after the machine is on again, we will find it runs the new kernel.

# cat /proc/version

 

 

 

2, install the necessary packages

 

# yum groupinstall "Development Libraries" "Development Tools"

# yum install transfig wget tar less texi2html libaio-devel dev86 glibc-devel e2fsprogs-devel gitk mkinitrd iasl xz-devel bzip2-devel cmake

# yum install pciutils-libs pciutils-devel SDL-devel libX11-devel gtk2-devel bridge-utils PyXML qemu-common qemu-img mercurial texinfo

# yum install libidn-devel yajl yajl-devel ocaml ocaml-findlib ocaml-findlib-devel python-devel uuid-devel libuuid-devel openssl-devel

# yum install glibc-devel.i686

 

if you are using Ubuntu, use this:

# apt-get install bcc bin86 gawk bridge-utils iproute libcurl3 libcurl4-openssl-dev bzip2 module-init-tools transfig tgif

# apt-get install texinfo texlive-latex-base texlive-latex-recommended texlive-fonts-extra texlive-fonts-recommended pciutils-dev mercurial

# apt-get install make gcc libc6-dev zlib1g-dev python python-dev python-twisted libncurses5-dev patch libvncserver-dev libsdl-dev libjpeg62-dev

# apt-get install iasl libbz2-dev e2fslibs-dev git-core uuid-dev ocaml ocaml-findlib libx11-dev bison flex xz-utils libyajl-dev

# apt-get install gettext libpixman-1-dev

 

or even simple:

# apt-get build-dep xen

 

 

3, download Xen source code

 

# wget http://bits.xensource.com/oss-xen/release/4.5.0/xen-4.5.0.tar.gz

# tar -zxvf xen-4.5.0.tar.gz

# cd xen-4.5.0

 

 

4, compile and install Xen

 

first time to compile

# ./configure

# make world

# make install

 

if recompile

# make dist

# make install

 

 

check the boot folder, you will find some new Xen related files:

 

 

# vim /boot/grub/grub.conf

 

Add the new boot information in this file, make the default boot sequence to Xen related title

 

 

 

Reboot the machine

# reboot

 

 

Test whether Xen is running

# xl li

 

if domain 0 shows, congratulations~
