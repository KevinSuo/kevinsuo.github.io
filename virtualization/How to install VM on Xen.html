How to install VM(PV) on Xen

 

Environment:

Linux environment: CentOS 6.5(Linux kernel 3.14)

Xen source code: Xen 4.5.0

Dom 0: CentOS 6.5

Dom U: CentOS 6.5

 

run as PV guest:

Yes:

Alpine Linux 2.3.x, 2.4.x;

CentOS 5, 6;

Debian Lenny, Squeeze;

Ubuntu 10.04, 11.04, 12.04;

Fedora 14, 16;

Oracle Linux 5, 6;

RHEL 5, 6;

SLES 10, 11;

OpenSUSE 11.4;

 

No:

Windows Vista, XP, 2008, 2003, 2000, 7;

 

 

Step:

 

1, create an image for VM

 

# dd if=/dev/mapper/vg_disco0007-lv_home of=/home/ksuo/Documents/centos65_pv.img oflag=direct bs=1M seek=40000 count=1

 

if: space source

of: img space

size: 40000*1M = 40G

 

change the directory to yours

 

 

2, create an configure file for VM

 

here is mine: centos65_pv.cfg

 

name = "centos65.pv"

vcpus = 2

memory = 4096

vif = [ 'mac= 00:16:3e:54:02:02, bridge=xenbr0' ]

disk = [ 'file:/home/ksuo/Documents/centos65_pv.img, xvda, w', 'file:/home/ksuo/Downloads/os_iso/CentOS-6.5-x86_64-bin-DVD1.iso, xvdc:cdrom, r' ]

 

# Kernel image to boot

kernel = "/home/ksuo/Documents/vmlinuz"        

 

# Optional

ramdisk = "/home/ksuo/Documents/initrd.img"

 

# Kernel command line options

extra = "root=/dev/xvda"

 

# open vncviewer

vfb=[ "type=vnc, vncpass=123456" ]

 

 

here are the links of OS iso files, vmlinuz, initrd.img

 

vmlinuz_http://vault.centos.org/6.4/os/x86_64/isolinux/vmlinuz

initrd.img_http://vault.centos.org/6.4/os/x86_64/isolinux/initrd.img

CentOS-6.5-x86_64-bin-DVD1.iso_http://centos.sonn.com/6.5/isos/x86_64/CentOS-6.5-x86_64-bin-DVD1.iso

 

 

3, start running the configure file

 

# xl create centos65_pv.cfg

 

use xl list to verify the vm has already been running

 

 

 

4, install the VM in GUI mode

 

#xl vncviewer 1 

1 is domu id

 

then install step by step

 

 

 

5, after installed guest OS to disk, modify the config file

 

 

name = "centos65.pv"

vcpus = 2

memory = 4096

vif = [ 'mac= 00:16:3e:54:02:02, bridge=xenbr0' ]

#change 1

#disk = [ 'file:/home/ksuo/Documents/centos65_pv.img, xvda, w', 'file:/home/ksuo/Downloads/os_iso/CentOS-6.5-x86_64-bin-DVD1.iso, xvdc:cdrom, r' ]

disk = [ 'file:/home/ksuo/Documents/centos65_pv.img, xvda, w']

 

# Kernel image to boot

#change 2

#kernel = "/home/ksuo/Documents/vmlinuz"        

bootloader = "/usr/local/bin/pygrub" 

 

# Optional

#change 3

#ramdisk = "/home/ksuo/Documents/initrd.img"

 

# Kernel command line options

extra = "root=/dev/xvda"

 

# open vncviewer

vfb=[ "type=vnc, vncpass=123456" ]

 

 

 

6, save the config file, run the VM

 

# xl create centos65_pv.cfg

# xl vncviewer 2

2 is the new id of domu

 

then configure the ip, dns, gateway, etc. Enjoy your new VM.
